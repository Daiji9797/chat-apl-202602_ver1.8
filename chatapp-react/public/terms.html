本利用規約は、本サービスを利用するすべてのユーザーに適用されます。本サービスの利用者（以下「ユーザー」）は、本規約に同意した上でサービスを利用するものとします。

本サービスは、以下の機能を提供するチャットアプリケーションです：

  • ユーザー認証・登録機能
  • AI対話機能（OpenAI API を利用）
  • チャットルーム管理
  • 目標達成支援機能
  • その他付随するすべての機能

第3条 ユーザーの責任

ユーザーは、以下の行為を禁止します：

提供者は、ユーザーに事前通知することなく、本サービスの機能追加・変更・中止を行う権利を有します。
